# Reports

Folder ini disiapkan untuk keluaran dokumentasi UAS:

- `final_report.pdf` atau `final_report.docx`
- `presentation.pptx`
- `evaluation/classification_report.csv`
- `evaluation/confusion_matrix.png`

Draft laporan teknis sebaiknya memuat pendahuluan, metodologi, hasil analisis, evaluasi model, kesimpulan, rekomendasi, dan referensi.
